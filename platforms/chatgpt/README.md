# ChatGPT Projects 셋업 가이드

ChatGPT Plus / Team / Enterprise 사용자라면 5분 안에 시작할 수 있다.
무료 사용자는 GPTs 또는 일반 대화에서 시스템 프롬프트를 직접 붙여 쓰는 방식으로 가능 (아래 "무료 사용자" 섹션 참고).

## 1. 새 프로젝트 만들기

ChatGPT 좌측 메뉴에서 **"새 프로젝트"** 클릭.

- **이름**: `오토라이팅 & 5F 회고`
- **설명**: `생각과 감정을 검열 없이 꺼내고, 3F/5F 회고로 디코딩하여 실행 가능한 다음 행동으로 연결하는 셀프코칭 프로젝트.`

## 2. 시스템 프롬프트 붙여넣기

`system-prompt.md` 안의 코드 블록 내용을 복사해서 프로젝트 **"지침"** 영역에 그대로 붙여넣는다.

## 3. 지식 파일 업로드

`knowledge-files/` 안의 3개 파일을 프로젝트에 업로드:

- `01-methodology.md` — 방법론 본문
- `02-emotion-words.md` — 감정 단어 목록
- `03-templates.md` — 템플릿 모음

## 4. 첫 대화

프로젝트 안에서 새 대화를 열고 다음을 입력한다.

```text
오토라이팅 시작.

오늘은 이 프로젝트의 첫 기록이야.
먼저 10분 동안 검열 없이 쓸 수 있게 시작 문장 3개를 제안해줘.
```

### 일반적인 평일 흐름

1. **"오토라이팅 시작"** → 시작 문장 받기 → 10–30분 자유 기록
2. **"3F 정리"** → 원문 붙여넣기 → Fact / Feeling / Finding 분리
3. 마지막에 AI가 묻는다: **"3F로 닫을까요, 5F까지 갈까요?"**
   - 소화가 목적이면 → "3F로 닫기"
   - 전환이 목적이면 → "5F까지"

## 5. 캘린더 등록

`../../calendar/blurbs.md`의 평일/주말 문구를 캘린더에 등록하면 루틴이 잘 잡힌다.

## 무료 사용자: GPTs 또는 일반 대화로 쓰기

ChatGPT 무료 플랜이면 Projects를 못 쓰지만, 다음 방법으로 비슷한 경험을 만들 수 있다.

**옵션 A — GPTs 만들기** (사용자 본인용)
1. ChatGPT에서 GPTs 생성 가능한 플랜이면 새 GPT를 만들고 `system-prompt.md`의 내용을 "Instructions"에 붙여넣는다.
2. `knowledge-files/` 3개를 Knowledge로 업로드한다.

**옵션 B — 일반 대화에서 매번 붙여넣기**
1. 새 대화를 열 때마다 첫 메시지로 `system-prompt.md` 내용을 붙여넣고, 이어서 "오토라이팅 시작"이라고 보낸다.
2. 감정 단어나 템플릿이 필요하면 그때그때 `docs/` 안의 해당 파일을 첨부한다.
3. 매번 시스템 프롬프트를 붙이는 게 번거로우면, 대화 한 개를 "메인 기록 대화"로 정해 그 안에서 모든 데일리 기록을 이어간다.

## 운영 리듬 (요약)

- **평일**: 3F 기본, 그날그날 5F까지 확장할지 선택
- **주말**: 한 주 기록을 모아 "주말 디코딩"
- **월간**: 주간 디코딩을 모아 "월간 리뷰"

자세한 방법론은 [docs/methodology.md](../../docs/methodology.md), 분기 가이드는 [docs/faq.md](../../docs/faq.md).
