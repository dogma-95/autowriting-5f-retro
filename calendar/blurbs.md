# 캘린더 등록용 문구

평일/주말 캘린더 일정에 그대로 복사해 쓰는 문구.

## 평일 캘린더

**제목**

```text
오토라이팅 3F
```

**설명**

```text
1. AI 도구 (ChatGPT / Claude / Gemini 중 셋업한 것) 열기
2. "오토라이팅 시작" 입력
3. 10~30분 동안 검열 없이 쓰기
4. "3F 정리" 입력 후 원문 붙여넣기
5. 오늘은 3F로 닫을지, 5F까지 갈지 선택
6. 필요하면 "3F로 닫기" 또는 "5F까지" 입력

성공 기준:
잘 썼는가가 아니라, 머릿속에 있던 것을 밖으로 꺼냈는가.
```

## 주말 캘린더

**제목**

```text
주말 디코딩 5F
```

**설명**

```text
1. 이번 주 데일리 기록 모으기
2. AI 도구 열기
3. "주말 디코딩" 입력
4. 이번 주 기록 붙여넣기
5. 반복 문제, 반복 감정, 숨겨진 자원 확인
6. 다음 주 Future Action 1~3개 도출
7. 다음 주 캘린더나 업무툴에 반영

성공 기준:
완벽한 분석이 아니라, 다음 주에 시도할 작은 행동 하나를 찾았는가.
```

## 월말 캘린더 (선택)

**제목**

```text
월간 리뷰
```

**설명**

```text
1. 이번 달 주간 디코딩 결과 모으기 (데일리 원문 X)
2. AI 도구 열기
3. "월간 리뷰" 입력
4. 이번 달 주간 디코딩 붙여넣기
5. 반복된 리더십 과제, 감정 패턴, 가진 자원 확인
6. 다음 달 실험할 행동 3개 도출
7. 유지할 루틴 / 버릴 루틴 정리

성공 기준:
한 달의 패턴을 한 줄로 설명할 수 있게 되었는가.
```
