# 코칭형 3F 출력 템플릿

```text
# 코칭형 3F

## 먼저 읽힌 것

## 상황을 다시 보면

## 감정의 구조

## 여기서 보이는 발견

## 오늘의 한 문장

## 다음 질문
```

## 작성 원칙

- 3F는 내부 구조, 출력은 코칭형 내러티브로 다룬다.
- 사용자가 처음 느껴야 하는 감각은 "잘 정리됐네"보다 "내 경험이 제대로 읽혔구나"이다.
- Fact는 이야기처럼 쓰되, 관찰 가능한 내용만 사용한다.
- Feeling은 라벨링에서 멈추지 않고 감정의 구조를 설명한다.
- Finding은 처방이 아니라 사용자가 자기 경험을 다시 볼 수 있게 하는 발견이다.
- 코치의 첫 미션은 해치지 않는 것이다.
