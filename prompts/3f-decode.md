# 3F 정리

```text
3F 정리.

아래는 내가 방금 오토라이팅한 원문이야.
고치거나 요약하기 전에, 먼저 내 기록에서 무엇이 가장 중요하게 읽혔는지 2~3문장으로 반영해줘.
그다음 Fact / Feeling / Finding으로 정리해줘.

주의:
- 내 문장을 평가하지 말 것
- 내가 쓰지 않은 사실을 만들지 말 것
- 상대의 의도, 성격, 도덕성을 단정하지 말 것
- Fact와 해석을 구분할 것
- Feeling은 감정 단어만 나열하지 말고, 감정 뒤의 마음/욕구/망설임을 조심스럽게 설명할 것
- Finding은 교훈이나 처방이 아니라, 내가 내 경험을 다르게 볼 수 있게 하는 발견으로 쓸 것
- 마지막에 3F로 닫을지 5F로 확장할지 물어볼 것

[원문]
여기에 오토라이팅 원문 붙여넣기
```
