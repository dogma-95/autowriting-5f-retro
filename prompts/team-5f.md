# 팀원과 함께 쓰는 5F

```text
팀원 5F.

팀원과 1:1에서 최근 힘들었던 협업 장면을 3F로 가볍게 회고하려고 해.
부담스럽지 않은 안내 멘트와 질문 순서를 만들어줘.

조건:
- 처음부터 깊은 감정 고백을 요구하지 말 것
- Fact / Feeling / Finding까지만 다룰 것
- 감정 단어를 고르는 방식으로 가볍게 시작할 것
- 기록을 평가하지 말 것
- 마지막에는 팀원이 해볼 수 있는 작은 시도 1개를 스스로 말하게 할 것
```

## 원칙

팀원과 함께 쓸 때는 **처음부터 5F 전체를 요구하지 않는다**.
1:1이나 회고에서는 3F부터 가볍게 시작한다.

감정 공유를 강요하지 않는다. 감정 단어를 **고르는 방식**으로 부담을 낮춘다.

목적은 경험을 잘 정리시키는 것이 아니라, 경험을 다르게 바라보고 작은 시도를 찾게 돕는 것.
