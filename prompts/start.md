# 오토라이팅 시작

## 풍부 모드 — 자유 기록부터 시작

```text
오토라이팅 시작.

오늘은 먼저 정리하지 않고 10분 동안 머릿속에 있는 것을 꺼내고 싶어.
시작 문장 3개와, 중간에 막혔을 때 이어갈 문장을 제안해줘.
```

## 유도 모드 — 막혔을 때

쓸 게 없거나 감정만 떠오를 때.

```text
오토라이팅 시작.

지금 막혔어. 질문으로 같이 끌어내줘.
한 번에 한 가지만 물어줘.
```

또는 짧은 편린만 던져도 AI가 자동으로 유도 모드로 들어간다.

```text
불안해서 미칠 것 같아요. 다 망친 것 같아요.
```

## 추론 모드 — 질문 답하기조차 어려울 때

```text
추론해줘. 지금 한 줄도 못 쓰겠어.
```

또는 편린 + 추론 요청.

```text
화가 났어. 다 그만두고 싶어. 이거로 3F 만들어줘.
```

## 언제 어느 모드?

| 사용자 상태 | 모드 |
|---|---|
| 10~30분 자유 기록 가능 | 풍부 모드 |
| 감정/생각이 흐릿하지만 질문에 답할 수 있음 | 유도 모드 |
| 짧은 한두 줄밖에 못 쓰겠음 | 추론 모드 |

세 모드는 한 대화 안에서 자유롭게 오갈 수 있다. AI가 사용자 상태를 보고 자동 전환하기도 하고, 사용자가 명시적으로 바꾸기도 한다.
