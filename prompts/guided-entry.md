# 유도 진입 (Guided Entry)

막혀 있을 때 AI가 질문으로 함께 끌어내주는 진입 방식.

## 기본 프롬프트

```text
지금 막혔어. 질문으로 유도해줘.

조건:
- 한 번에 한 가지만 물어줘.
- 가장 무거운 감정부터 시작해서, 그 감정이 강해진 한 장면, 그 장면의 사실 순서로.
- 내가 "모르겠어요"라고 답하면 더 묻지 말고 추론 모드로 전환해줘.
- 4번째 질문쯤에 "이 정도면 3F로 정리해볼까요?"라고 제안해줘.
```

## 편린만 던지고 시작하는 방식

길게 쓸 필요 없다. 한두 줄만 던지면 AI가 자동으로 유도 모드로 들어간다.

```text
불안해서 미칠 것 같아요.
```

```text
오늘 너무 답답해. 왜인지 모르겠어.
```

```text
화가 났어. 누구한테인지도 모르겠어.
```

## 진행 흐름 (참고)

AI는 다음 순서로 한 가지씩 물어준다.

1. **가장 무거운 감정 한 단어**
   - 6 카테고리(편안·기쁨·감사·자신감·만족 / 불안·답답함·피로·슬픔·화·혼란·관계적 어려움)에서 가까운 단어 제안
2. **그 감정이 가장 강해진 한 장면**
   - 떠올릴 수 있어요? 안 떠올라도 괜찮아요.
3. **그 장면의 사실 한 조각**
   - 누가 있었는지 / 어떤 말이 오갔는지 / 어디서였는지 중 하나
4. **더 추가하고 싶은 한 가지**
5. AI가 "이 정도면 같이 3F로 정리해볼까요?" 제안

## 빠져나오는 길

- "이제 됐어요" / "정리해줘" → 즉시 3F 정리로 전환
- "추론해줘" / "그냥 만들어줘" → 추론 모드로 전환
- "모르겠어요" / "기억 안 나요" → AI가 자동으로 추론 모드로 전환

답하기 어려운 질문이 나오면 그냥 "모르겠어요"라고 답해도 된다. 도구가 또 하나의 과업이 되면 안 된다.

## 원칙

- 유도 모드는 도구가 사용자에게 맞춰주는 모드다. 사용자가 도구에 맞출 필요 없다.
- AI가 두 가지를 한 번에 물으면 "한 가지만"이라고 한 번 짚어준다.
- 한 가지도 답하기 어려운 날엔 `prompts/3f-decode.md`로 가지 말고 "추론해줘"로 곧장.
